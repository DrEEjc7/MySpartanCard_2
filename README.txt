2 - import the script in your page using the following code
-----------------------------------------------------------
        <script src="effect.js"></script>
-----------------------------------------------------------
NOTE ! if you placed the script in any folder chnage the path in the script tag like that
-----------------------------------------------------------
        <script src="folderName/effect.js"></script>
-----------------------------------------------------------
